## Overview
Shex-Libretro is a simple RetroArch core designed to execute shell scripts for launching games. "Shex" stands for "Shell Execution," reflecting its straightforward functionality.

This project was created to improve the game launching experience on Linux, especially for users running games with Wine alongside RetroArch for retro games.

**Key Features:**
- Launch games using shell scripts.
- Lightweight core written in C.

## Tech Stack
- **Language:** C
- **Platform:** RetroArch Core

## Features

### 1. Simple Game Launching
Shex-Libretro integrates with RetroArch to run shell scripts, allowing users to launch games directly from their library.
