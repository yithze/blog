## Overview
Rize Cafe is a comprehensive cashier application designed to streamline transaction management and table operations for cafe businesses. Built with Laravel 11 and SQLite, this project introduces an intuitive system for both cashiers and managers to handle daily operations efficiently.

**Key Features:**
- Transaction processing with table management.
- Clear-table mechanism ensuring no table can be reused until cleared.
- Invoice generation and printing.
- Role-specific functionalities for cashiers and managers.


## Tech Stack
- **Framework:** Laravel 11
- **Database:** SQLite
- **Additional Tools:** Blade Templates, JavaScript

![Rize-Cashier](https://github.com/user-attachments/assets/a85fbc13-28b1-41fa-af6a-223f0a095181)

## Features in Detail

### 1. Transaction Management
Cashiers can create transactions by selecting menu items, inputting customer names, assigning tables, and calculating change dynamically. Tables are managed in a way that ensures no table can be reused until cleared.

**Screenshot:**
![Rize-Cashier](https://github.com/user-attachments/assets/a85fbc13-28b1-41fa-af6a-223f0a095181)

### 2. History and Invoice
Cashiers can:
- View transaction history for quick reference.
- Generate and print invoices directly from the system.

**Screenshots:**
![Rize-Transaction](https://github.com/user-attachments/assets/eecc043b-ec40-4769-854c-d00b9b4add48)

![Rize-Invoice](https://github.com/user-attachments/assets/d5222005-012c-4e22-90c6-648dcf95c203)

### 3. Table Management
During transactions, cashiers assign tables. Tables in use cannot be reassigned until they are cleared.

### 4. Menu and Income Management (Manager Role)
Managers have access to:
- Managing menu items (add, edit, delete).
- Viewing transaction and income history with a draggable calendar filter for custom date ranges.
