## Overview
OurDoc is a user-friendly document management application tailored for individuals and businesses seeking efficient solutions for file management, table editing, and document approval workflows. Built with Laravel 11 and SQLite, the system delivers simplicity and functionality for various use cases.

**Key Features:**
- File management with preview capability.
- Table editor for simplified data manipulation akin to Excel.
- Document approval workflows with custom approval lists.
- Intuitive dashboard for quick navigation and overview.

## Tech Stack
- **Framework:** Laravel 11
- **Database:** SQLite
- **Additional Tools:** Blade Templates, JavaScript

## Features in Detail

### 1. File Management
Users can manage documents efficiently by organizing them into lists. The system also supports file preview to enable quick reference without requiring downloads.
